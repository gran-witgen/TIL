# container

コンテナ  

グリッドシステムを利用する際は<body>の直下に配置すること  

コンテナには2種類ある  

1. 固定幅コンテナ  
2. 可変幅コンテナ  

固定幅コンテナ  

ブレイクポイントで幅で変化  

```html
<body>
  <div class="container">
    <h1>hello></h1>
  </div>
</body>
```

可変幅コンテナ  

ブラウザ横いっぱいに広がる  

```html
<body>
  <div class="container-fluid">
    <h1>hello</h1>
  </div>
</body>
```
